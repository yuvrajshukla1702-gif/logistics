# FleetPulse — Shipment Tracking Web Application

FleetPulse is a modern, responsive web application for tracking shipments and monitoring delivery progress in one place. It helps users view shipment status, route checkpoints, estimated delivery time, and fleet activity through a clean dashboard-style interface.

## Features

- **Shipment Tracker** — Search and view shipment details, progress, and route milestones
- **Dashboard** — Add, update, and manage shipments from an admin-style panel
- **Live Simulation Mode** — Demo real-time-like status and progress updates
- **Dark / Light Theme** — User preference saved in the browser
- **Responsive UI** — Works on mobile and desktop using Tailwind CSS
- **Client-side Persistence** — Shipment data stored in `localStorage` for refresh-safe demo use

## Tech Stack

**Frontend**
- React 19
- Vite
- Tailwind CSS v4
- React Router DOM

**Backend (in progress)**
- Node.js
- Express.js
- CORS enabled for frontend integration

## Project Status

This is a learning/portfolio project. The **frontend is functional** and demo-ready. The **backend is currently a starter server** and will be extended with REST APIs and database integration.
